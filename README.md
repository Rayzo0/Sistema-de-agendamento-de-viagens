### Sobre o projeto

Uma plataforma direcionada a realização de agendamento de viagens.
Desenvolvida em C# com ASPNET-CORE 3.1 e SQL SERVER. O Projeto Sul Fluminense Airlines foi desenvolvido pelos
alunos Guilheme Malaquias, Vitor Cardoso e Victor Martins do curso de sistemas de informação da Universidadde Geraldo Di Biase 
para a avaliação do segundo bimestre da disciplina linguagem técnica de programação 3 que é ministrada pelo professor Rosenclever Gazoni.
